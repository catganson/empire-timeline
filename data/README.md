# Economic Data Snapshots

Run:

```bash
./scripts/update-economic-data.sh
```

This writes official CSV snapshots used by `index.html`:
- `current-account.csv` (A124RC1A027NBEA)
- `gdp.csv` (GDPA)
- `niip.csv` (IIPUSNETIA)
- `foreign-treasury-holdings.csv` (ROWTSEA027N)
- `exports-goods.csv` (BOPXG)
- `exports-services.csv` (BOPXS)
